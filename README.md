# はじめかた

1. Node.js と VSCode と VSCode 拡張の Prettier を入れる
2. ターミナルを開く
3. `npm i` する
4. `npm run dev` すると `http://localhost:1234` にプレビューされる
5. src 内のファイルを編集しながらゲームを作る

## ゲームをデプロイする

1. ターミナルを開く
2. `npm run build` する
3. `dist` フォルダに出力されるので、それを公開する
